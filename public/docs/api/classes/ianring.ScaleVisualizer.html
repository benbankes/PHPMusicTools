normalizeScalePitches()

normalizeScalePitches(array<mixed,\ianring\Pitch>  $pitches) : array<mixed,\ianring\Pitch>

What this function has got to do is make sure that the C sharp major scale uses an E sharp, not an F natural. To do that it recognizes scales that are diatonic, and forces each note to be on consecutive steps. To do that, it assumes that the first note is on the correct step!

TODO: it should handle complex scales like bebop and octotonic properly. Good luck!

returns true if the scale is chiral.

Chirality means that something is distinguishable from its reflection, and can't be transformed into its reflection by rotation. see: https://en.wikipedia.org/wiki/Chirality_(mathematics) and http://ianring.com/scales

getChordBitMasks()

getChordBitMasks()

This method constructs tertiary triads built on each member of the scale.

For example when given a major scale, this should return

scale: 101010110101 [ 10010001, (root tonic triad) 1000100100, (minor triad on the second degree) 100010010000, (minor triad on the third degree) 1 001000100000, (major triad on the fourth degree) 100 100010000000, (major triad on the fifth degree) 10001 001000000000, (minor triad on the sixth degree) 100100 100000000000, (dim triad on the seventh degree) ]

getTones()

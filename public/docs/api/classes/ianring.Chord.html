toVexFlow()

toVexFlow(  $duration = 'w')

renders a script fragment for a chord. What we want to end up with is a string that looks like this:

new Vex.Flow.StaveNote({ keys: ["c#/4","e/4","g#/4"], duration: "w" }) .addAccidental(0, new Vex.Flow.Accidental("#")) .addAccidental(2, new Vex.Flow.Accidental("#"));

(^ that's a c sharp minor triad) the "key" part puts the note head in place. "addAccidental" is needed if you want the accidental to be shown; vexflow doesn't render them automatically because they could be present earlier in the measure or in the key signature.

Parameters

$duration
